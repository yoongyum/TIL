1973

GNU
Gnu is Not Unix

kernel
하드웨어와 응용프로그램을 이어주는 운영체제의 핵심 시스템소프트웨어

Linus Torvalds
헬싱키 대학생이던 리누스 토발즈는 앤디 타넨바움의 MINIX를 개조한 Linux 발표

Linux
리누스 토발즈가 작성한 커널, GNU 프로젝트의 라이브러리와 도구가 포함된 운영체제
Redhat, Debian, Ubuntu, Android 등 다양한 배포판이 존재

Shell
운영체제의 커널과 사용자를 이어주는 소프트웨어
bash: brian Fox가 작성한 유닉스 쉘, 다양한 운영체제에서 기본 쉘로 채택
zsh: paul Falstad가 작성한 유닉스 쉘, 현재까지 가장 완벽한 쉘

-al
--help 도움말
pwd 현재위치 
touch 파일생성
* 에스터리스크 =모든이라는 뜻
rm -r 디렉토리삭제
vim 텍스트 에디터
esc 노멀모드로 돌아간다.
