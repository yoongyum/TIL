# git push


```shell
$ git add {file name}
$ git commit
$ git push origin master
```


## Commit Convention


```shell
feat: features
docs: documentations
conf: configurations
test: test
fix: bug- fix
refactor: refactoring
ci: Continuous Integration
build: Build
perf: Performance
```


## init


```shell
$ git init
$ git remote add origin {url}
$ git branch -M main
```
